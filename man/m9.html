8.Управління договорами .

Управління договорами здійснюється у вкладці "Договора" див. мал. 8.1. Таблиця ліворуч містить усі наявні логовори у вигляді списку. Контекстне меню дозволяє додавати новий договір, редагувати та видаляти один із наявних договорів. Таблиця праворуч містить ціни для кожної позиціі з каталогу згідно договору який виділено у списку договорів. Коли ви створюєте новий договір, його список цін порожній. Ці ціни, для конкретного договору, можно завантажити із файла меню-раскладки Excel, лібо завантажити усі позиціі каталогу та внести ціни вручну. В списку договорів є тестовий договір, которий не можете бути вилучений. Цей договір потрібен для коректного створення накладної надходження. Після відкриття форми для нового надходження (див. п. 3) тестовий договір є договором для нової накладної. Ви можете змінити тестовий договір на інший, використвуючи випадаючий список, після цього всі ціни в накладної перерахуються згідно цінам з договору. Вибір договіра для накладної надходження зберігається у файлі БД. Вибір договору також доступний у витратній накладній, але там він служить тільки для вставки цін витрати та цей вибір договору не зберігається у витратній накладній. Для формування звіту по договорам можно викроститовувати звіти по руху товарів, там є відповідні опціі для вибору договорів.
