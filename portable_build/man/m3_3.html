3.3.Автоматичний імпорт надходження з файлу Excel.

Для тих, хто веде продовольчу службу використовуючи меню розкладки, можливо виконувати автоматичний імпорт прибуткової накладної із файлу Excel. Програма за замовчуванням налаштована на імпорт із файлу певного виду, але імпорт можливий також із будь-якого Excel файлу, якщо він містить рядки, що містять колонки з кодами з каталогу, найменуванням та кількістю. При цьому потрібно ручне введення індексів імпорту, про що буде розказано пізніше. Певний вид файлу накладної, на який програма налаштована за умовчанням, показаний на мал. 3.7. По суті, це накладна, яка має вкладки по днях, і вкладку “Загальна”, зроблені на основі меню-розкладки. Саме із вкладки “Загальна” і здійснюється автоматичне зчитування даних.

Мал.3.7 Накладний файл Excel для якого імпорт можливий без переналаштування індексів імпорту.

Послідовність дій для автоматичного імпорту надходження описана на мал. 3.8.

1.1.Відкрийте контекстне меню правою кнопкою миші на таблиці накладних. Виберіть пункт "Імпорт надходження з Excel".

2.У діалоговому вікні імпорту, що відкрилося, введіть:

1 – номер накладної

2 – дату надходження

3 – постачальника

4 – виберіть файл накладної на диску, для цього скористайтеся кнопкою … для перегляду файлової системи.

Потім натисніть кнопку “почати імпорт”. Нова накладна автоматично завантажить дані із файлу і стане на прибуток.
