6.5.Звіт “Рух води для контрагента”.

Звіт про рух води дозволяє проводити порівняльний аналіз: скільки води було реально видано і скільки мало бути використане за нормою. Для того щоб розрахунок за нормою був правильним, кожного тижня за вказаний період має бути одна накладна видачі (не обов'язково води) із зазначеною кількістю людей. Програма автоматично братиме цю кількість людей для розрахунку. Інші накладні на видачу цьому контрагента в межах тижня повинні містити нулі для кількості людей, в протилежному випадку кількість людей буде просумовано в звіті. Кількість людей за конкретний тиждень вказується для одного дня, тобто звіт рахує що вказана кількість буде однаковою для всіх днів тижня. Дата накладної в якій вказується кількість людей для конкретного тижня, може бути будь-якою в рамках цього тижня. Зовнішній вигляд звіту показано на мал. 6.8. Звіт працює лише з одним кодом для води за каталогом – 4012.
